jQuery DOM 클레스

문서객체모델의 class 조작은 사용자가 선택한 요소에 클래스 서식을 추가하거나 삭제 또는 클래스가 없는 요소를 찾아서 서식을 적용하는 메서드이다.

  • addClass : 선택한 요소에 클래스를 추가
  • removeClass : 선택한 요소에 클래스를 제거
  • toggleClass : 선택한 요소에 서식을 추가하거나 / 제거
  • hasClass : 선택한 요소 클래스 이름이 있는지 여부를 판단하여 true/false(boolean)표시
  • css : 사용자가 선택한 요소에 스타일 서식 적용
