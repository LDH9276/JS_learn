자바스크립트 연산자

자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할 때 사용하는 기호이다.

  • 산술연산자 : +, -, *, /, %(나머지)
  • 문자열 결합 연산자 : +(문자 + 숫자, 문자 + 문자 일 경우)
  • 비교연산자 : >, <, >=, <=, ==(값이 같다), ===(값, 형식 모두 같을 경우)
  • 대입 연산자 : =, +=, -=
  • 중감 연산자 : ++(1씩 증가), --(1씩 감소)
  • 논리연산자 : &&, ||, !=
  • 상황조건연산자 : 조건식 ?참일때 값 거짓일 때 값
  • 타입연산자 : typeof
