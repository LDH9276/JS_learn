변수 복습과 유의사항

  • var : es6 이전 사용되는 변수 / 단 자주 바뀌는 문제점으로 es6이후로는 사용 x
  • let : es6 이후 문법으로 새롭게 나온 변수 키워드 / 변하는 값을 저장할 때 사용한다.
  • const : es6 이후 문법으로 새롭게 나온 변수 키워드 / 변하지 않는 값을 저장할 때 사용한다.

var의 경우, 변수선언을 여러번 해도 오류가 나지 않는다. (유지관리성에서 문제가 많다.)

  1. 전역변수는 초소한으로 사용한다.
  2. 변수는 함수와 시작부분에서 선언한다.
  3. for문에서는 카운터 변수를 선언할때 var예약어를 사용하지 않아야 한다.

    var i;
    for({)i=1; 1<=10; i++){
      출력할 내용
    }

    for(let i=1; i<=10; i++){
      출력할 내용
    }
