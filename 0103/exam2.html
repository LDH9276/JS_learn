02_document(문서객체모델)_object

DOM(Document Object Model은 넓은 의미로는 웹브라우저가 html을 인식하는 방법이고 좁의 의미로는 웹 문사안에 있는 요소(객체)들의 집합이다.

웹브라우저(크롬, 사파리, 파이어폭스, 오페라, 익스플로러, edge, 모바일 브라우저)마다 DOM구현방식이 호환되지 않아서 W3C에서 DOM에 대한 표준규칙을 작성하였다.

2. document 객체의 속성

  • domain : 현재 문서가 있는 서버의 도메인 이름
  • tite : 현재 문서의 제목
  • bgColor : 문서의 배경색
  • fgColor : 문서의 글자색
  • linkColor : 문서의 링크색
  • alinkColor : 링크를 클릭하였을 때 색
  • vlinkColor : 방문했던 링크의 글자색
  • anchors : 문서의 anchor들의 배열
  • cookie : 쿠키에 대한 정보
  • url : 문서의 url 주소

3. document객체의 메소드

  • open() : 데이터를 보낼 준비
  • close() : 문서에 데이터 출력을 마무리함
  • clear() : 브라우저에서 문서를 지움
  • write() : 문서에 데이터를 출력함
  • writeIn : 문서에 줄 바꾸기를 포함하여 데이터를 출력함
