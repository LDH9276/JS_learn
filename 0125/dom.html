제이쿼리 문서객체 관련 메서드

문서객체 관련 메서드란? DOM에서 HTML문서 안에서 모든 요소들의 내용물을 제이쿼리를 사용하여 글자를 변경하거나 추가, 삭제, 속성값을 변경할 수 있다.

https://oscarotero.com/jquery 페이지에서 API attributes, Manipulation 카테고리에서 확인 가능.

  • $('대상').text(); - html요소를 선택하여 내용을 입력하거나 가져오기 위한 메소드 (==.document.getElementById.innerText="";
  • $('대상').html(); - html요소를 선택하여 html태그요소를 추가하거나 가져오기 위한 메소드(==document.getElementById.innerHTML="";
  • $('대상').attr(); - html요소를 선택하여 속성값을 추가하거나 변경하기 위한 메서드
  • $('대상').val(); - html요소를 선택하여 input요소의 속성값을 추가하거나 변경하기 위한 메서드
  • $('대상').empty(); - 내용지우기
  • $('대상').remove(); - 요소지우기
  • $('대상').addClass(); - 클래스 추가하기
  • $('대상').removeClass(); - 클래스 제거하기
  • $('대상').hasClass(); - 클래스 존재여부확인
  • $('대상').toggleClass(); - 클래스 토글 적용
  • $('대상').appendTo(); - 부모내용 마지막 위치로 이동하기
  • $('대상').prependTo(); - 부모내용 처음 위치로 이동하기

실습 1 : 버튼을 클릭하면 선택한 요소의 문서내용을 삽입하거나 내용이 변경된다.

아래 text() 버튼을 클릭하면 이곳의 텍스트 내용이 변경됩니다.

아래 html() 버튼을 클릭하면 이곳의 html요소가 추가되어 변경됩니다.


실습 2 : 버튼을 클릭하면 선택한 이미지 요소의 속성값을 변경하여 다른 이미지가 나오게 하기

네이버
  • 현대랜탈케어 페이스북 이벤트
  • 현대백화점 공식 블로그 리뉴얼 오픈이벤트
  • 현대모터스튜디오서울 커피클래스 5월 참가자모집
  • 현대랜탈케어 큐밍 더 공기청정기

$('대상').attr('속성') : 사용자가 선택한 요소의 속성값을 가져온다.

$('대상').attr('속성', '값') : 사용자가 선택한 요소의 속성값을 변경한다.

$('대상').attr({'속성':'값'}, {'속성':'값'} : 사용자가 선택한 속성들에 값을 대입하여 변경한다.
