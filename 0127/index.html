제이쿼리 문서객체 조작 - 포트폴리오 이미지 갤러리

12장의 작품 이미지를 사용하여 갤러리 페이지 구현하기

  • 내비게이션 메뉴 5개를 만들고 아래 12장의 이미지 목록을 가로 4장씩 3줄로 배치
  • 메뉴 클릭시 해당 주제에 맞는 이미지만 화면에 보이게 한다
  • 메뉴 클릭시 해당 메뉴에 서식을 적용하여 현재 상태를 표시한다
  • 갤러리 이미지에 마우스 오버 시 캡션이 나오도록 한다
  • 이미지 클릭시 해당 경로의 이미지가 모달 윈도창으로 뜨도록 한다
  • 모달 윈도에서 닫기 버튼을 클릭하면 사라지도록 한다
  • 모달 윈도에서 이전, 다음 버튼 클릭시 해당 사진의 다음, 이전 사진이 나도록 한다
  • 모달 윈도에서 이미지의 하단에 캡션 내용이 표시되도록 한다
  • 모달 윈도에서 이미지 하단에 페이지 번호가 출력되도록 한다
  • 이미지에 마우스 오버 시 다양한 효과를 적용한다.
