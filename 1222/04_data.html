02 데이터 타입(유형) p.468

자바스크립트에서 사용하는 데이터 유형에 대해 알아보기

  • 문자 데이터형 (string)
  • 숫자 데이터형 (number)
  • 불린형 (boolean) true(참)/false(거짓)
  • 찾을 수 없는 데이터형(undefined)
  • 값이 없는 데이터(null)
  • 배열형 데이터 (array)
  • 함수형 데이터 (function)
  • 객체(object) document/window/alert/confirm...

typeof 연산자는 해당 데이터 타입이 무엇인지 나타난다. 각 데이터들은 원시형(기본형) 데이터라고 부른다.
